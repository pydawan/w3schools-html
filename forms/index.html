HTML Forms

O elemento <form>

  1. O elemento <form> é utilizado para coletar dados de entrada do usuário.
  2. Um formulário HTML contém elementos de formulário.
  3. Elementos de formulário são diferentes tipos de elementos de entrada como campos de texto (text fiedls),
    caixas de marcação de múltiplas opções (checkboxes), caixas de marcação de opção única, botões de envio (submit buttons) e mais.
  4. O elemento <input> é o elemento mais importante do formulário.
  5. O elemento <input> pode ser exibido de várias maneiras dependendo do atributo type.

Seguem alguns exemplos:

Tipo (type) Descrição
<input type="text"> Define um campo de entrada de texto de uma linha.
<input type="radio"> Define um radio button (para seleção de uma entre várias opções)
<input type="submit"> Define um submit button (para envio do formulário)

The Submit Button

<input type="submit"> define um botão para enviar os dados do formulário para um manipulador de formulário (form-handler) ou melhor dizendo, um processador de formulário.

O form-handler tipicamente é uma página do servidor com um script para processamento de dados de entrada.

O form-handler é especificado no atributo action do formulário.
