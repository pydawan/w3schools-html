URL - Uniform Resource Locator

Uma URL (Uniform Resource Locator) é utilizada para endereçar um documento (ou outro dado) na web.

Um endereço web como https://www.w3schools.com/html/default.asp segue estas regras sintáticas:
scheme://prefix.domain:port/path/filename

Explicação:

  • scheme - define um tipo de serviço Internet (http ou https são os mais comuns)
  • prefix - define um prefixo de domínio (o padrão do http é o www)
  • domain - define o nome do domínio na Internet (como w3schools.com)
  • port - define o número da porta no hospedeiro (host)
  • path - define um caminho (path) para o servidor (se omitido: o diretório raiz do site)
  • filename - define um nome de um documento ou recurso
